# Cloud Pipeline

No local computer is required for normal operation.

Flow: Google Sheet -> Supabase -> Vercel Admin -> Review -> Xyla -> Shopify -> Metrics -> Slack.

Rules:
- GitHub governs system files.
- Drive stores assets and architecture docs.
- Marketing Sheet controls content dates and formats.
- Supabase stores operating state.
- Vercel displays admin workflows.
- Shopify owns conversion.
- Slack handles alerts.
- Xyla receives reviewed social content only.
- Primary timezone is America/New_York.
